# S3-Compatible Storage with Zstd Compression POC

## Project Overview

This repository contains a Proof of Concept (POC) demonstrating how to implement S3-compatible storage with zstd compression functionality.

## Запуск

-  Установит зависимости:
  ```bash
  pip3 install -r requirements.txt
  ```

- создайте .env файл и заполните его данными:

   ```bash
   mv .env.example .env   
   ```

- Запустите скрипт: 
  ```bash
  python3 poc_s3_zstd.py
  ```
